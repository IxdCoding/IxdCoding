发表于 2019-09-03

MarkdownPad是Windows下的一个全功能Markdown编辑器。

Markdown Pad可以理解为一个自定义的页面，下面我们来说说Markdown Pad的基本使用方法。

列表

有序列表

有序列表是行首使用数字+”.”后接一个半角空格符。如下所示：

  1. xxx
  2. xxx

效果如下：

  1. 国立三闾大学副教授
  2. 克莱登大学肄业

无序列表

无序列表是行首使用”*”，后接一个半角空格符。如下所示：

  • 国立三闾大学副教授

  • 克莱登大学肄业

效果如下：

  • 国立三闾大学
  • 克莱登大学肄业

有序列表和无序列表均支持不同层级的缩进，而且不同的解析器会给无序列表的不同层级分配不同的列表标示符。

有序列表和无序列表还可以混用，形成更加有层次感的列表。
